# Markdown Previewer

An old project built in the first half of 2025 as the start of my journey and posted here to attach to the course's page and to preserve my path.

This project is a part of FreeCodeCamp's Front End Development Libraries Certification. The goal is to build a Markdown Previewer that allows users to write Markdown in an editor and see the rendered HTML output in real-time.

**User Stories:**
- [x] `textarea` element with `id="editor"`
- [x] Element with `id="preview"` to display rendered HTML
- [x] `#preview` updates in real-time as text is entered into `#editor`
- [x] GitHub flavored markdown is rendered as HTML (using Marked library)
- [x] Default text contains valid markdown with: H1, H2, link, inline code, code block, list item, blockquote, image, bolded text
- [x] Default markdown is rendered as HTML in `#preview` on first load
- [x] Bonus: Carriage returns render as `<br>` elements (optional)
